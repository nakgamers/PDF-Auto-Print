# Nandz PDF Auto Print

Modern batch PDF printing tool for Windows.

![Preview](preview.png)

## Features

- Batch PDF printing
- Printer dropdown selection
- Modern GUI
- Real-time logging
- Portable SumatraPDF engine
- Windows installer

## Requirements

- Windows 10 / 11
